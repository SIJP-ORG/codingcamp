SIJP コーディング キャンプへようこそ！

小学生高学年(5年生以上）から中学生を対象にした「初心者向け コーディング キャンプ JavaScript編」を開催します。インストラクターとしてSIJP社会人スタッフとSIJP学生部スタッフが丁寧に、そして楽しく、指導いたします。JavaScriptは主にWebブラウザで動作するプログラミング言語で、もっともポピュラーなプログラミング言語の一つです。普段何気なくWebサイトを閲覧している時も、実は裏でJavaScriptがいろいろな機能を実現しています。

キャンプの雰囲気

セッションは「キャンプ」と呼び、あえて「クラス」とは呼びません。参加している人たち全員で協力しあいながら、プログラミングやコーディングを体験します。分からないことや困ったことがあれば、インストラクターやアシスタント・スタッフが教えてくれます。一緒に参加する他のお友達に助けてもらったりしても構いません。


[2016年6月18日 JavaScript コーディング キャンプ]

参加者それぞれにノートパソコン一台が必要です

  • WindowsもしくはMacノートパソコン
  • 必ずGoogle Chrome ブラウザをあらかじめインストールしておいてください
  • 電源アダプター（電源の確保は保証できません。ノートパソコンは充電をフルにしておいてください）
  • USBドライブを読み込める状態を確認しておいてください（とくに最新のMacなどはアダプター等が必要かもしれません）
  • 日本語を入力できる環境に設定されていることを確認しておいてください

出来る範囲で、以下の点を当日までに確認しください

  • お子様に、英語と日本語の入力の切り替えの方法を教えておいてください
  • プログラミングでは英語のアルファベット以外の記号文字等を使用します（例：[ や / など）。キーの位置をご確認ください
  • プログラミングではコピー＆ぺイスト（切り取り＆貼り付け）が便利です。Ctrl+Cなどのショートカットをご確認ください
  • Windows OSではファイルの拡張子（.txtや.html等）が表示されないモードになっている場合が多く混乱しやすいかもしれません。Windows10でファイルの拡張子を表示する方法
